# 🔔 Core Notifications Checklist

## 🧩 Account & Access

* [x] Account created (welcome / confirmation)
* [x] Login from new device/location
* [x] Password changed
* [x] Email changed
* [x] Unauthorized access attempt blocked

---

## 📖 Timeline Activity

* [x] Timeline created
* [x] Timeline updated (significant changes only)
* [x] Timeline deleted (confirmation)

---

## 🧑‍🤝‍🧑 Ownership & Access Control

* [x] Invited to timeline
* [x] Access granted
* [x] Access revoked
* [x] Role/permission changed

---

## 🔄 Sync & Integrations (GitHub etc.)

* [x] Integration connected
* [x] Integration disconnected
* [x] Manual sync completed
* [x] Auto-sync success (only if meaningful updates)
* [x] Sync failed (action required)

---

## 📊 Milestones & Progress

* [x] Milestone created
* [x] Milestone completed
* [x] Significant progress update (optional digest)

---

## ⚠️ Errors & System Events

* [x] Critical failure (sync / pipeline / save failure)
* [x] Data conflict detected
* [x] Action required to continue (e.g. expired token)

---

## 🔐 Security & Privacy

* [x] Password reset requested
* [x] Suspicious activity detected
* [x] Privacy setting changed (e.g. public → private)
* [x] Passphrase created/updated (for private timelines)

---

## 💳 Billing (if using paid plans)

* [x] Subscription started
* [x] Payment successful
* [x] Payment failed
* [x] Subscription expiring
* [x] Subscription cancelled
* [x] Plan upgraded/downgraded

---

## 📬 System / Product (Minimal Only)

* [x] Important feature release (rare)
* [x] Policy / terms update (required)

---

# 🚫 Explicitly Avoid (Noise)

* ❌ Every small edit/update
* ❌ Every commit synced
* ❌ Generic “activity happened” spam
* ❌ Marketing blasts disguised as notifications

---

# ⚡ Delivery Channels (keep tight)

* In-app → default for most
* Email → only for:

  * access/security
  * billing
  * failures
